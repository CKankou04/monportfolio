Formation Aux Situations d'Urgences

Ce Projet a été réalisé dans le cadre des études et par groupe de 5 à 7 personnes.
 
Résumé:
Le CESU est l’unite de formation des SAMU. Ils foment le personnel de la santé à la gestion des situations sanitaires exceptionnelles. Cette formation s’adresse aux décideurs notamment les médecins régulateur, les chefs des pompiers etc. dont le but est d’organiser les secours pour évacuer les victimes vers les centres de soin, au plus vite et dans les meilleures conditions. L’un des outils pour cette formation est un jeu de plateau.
Le projet consistait à modéliser et développer une plateforme de jeu multi-joueurs qui permet de gérer l’inscription des joueurs à la plateforme, la connexion des joueurs, l’affectation des rôles, la disposition d’un système de communication entre joueurs, La représentation graphique du jeu et le paramétrage l’état des victimes.
 
Technologies Utilisées:
HTML - CSS - JavaScript - AJAX - jquery - Three.js - l'Architecture MVC - Base de Donnes MySQL - Serveur WAMP
